# Loan Default Prediction

## 📌 Problem
Predict whether a customer will default on a loan.

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Seaborn / Matplotlib
- Scikit-learn

## 📊 Steps
- Data Cleaning
- EDA
- Feature Engineering
- Model Building

## 🤖 Models Used
- Logistic Regression
- Decision Tree

## 📈 Result
- Achieved good prediction accuracy
- Identified key factors affecting loan default

## 📌 Conclusion
Machine learning can help banks reduce risk by identifying high-risk customers.
