# 🏦 Home Loan Default Prediction

## 📌 Project Overview
This project aims to predict whether a customer will default on a home loan using machine learning techniques. By analyzing customer financial and demographic data, the model helps identify high-risk applicants, enabling better decision-making for financial institutions.

---

## 🎯 Problem Statement
Loan defaults are a major risk for banks and financial institutions. The objective of this project is to:
- Analyze customer data
- Identify key factors influencing loan default
- Build a predictive model to classify customers as defaulters or non-defaulters

---

## 📂 Dataset
- Dataset contains customer details such as income, loan amount, credit history, and other financial attributes
- Target variable: Loan Default (Yes/No)

---

## 🔍 Exploratory Data Analysis (EDA)

Performed analysis including:
- Data structure and summary statistics
- Handling missing values
- Checking duplicates
- Distribution analysis of key variables

### 📊 Key Insights:
- Credit history is a major factor influencing loan approval/default
- Higher income customers tend to have lower default risk
- Loan amount and applicant income show strong relationships
- Some imbalance observed in target variable

## 📌 Conclusion

This project demonstrates how machine learning can be used to reduce financial risk by predicting loan defaults.

Key takeaways:

- Data-driven models can assist in better loan approval decisions
- Helps banks minimize losses and improve risk management
